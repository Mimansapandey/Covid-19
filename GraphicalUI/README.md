# General info
A cross platform React project to get latest updated on the current situation of the pandemic.

# Prerequisities
```
Expo
npm install -g expo-cli
```
## Installing locally using expo
```
cd Covid-19-master
expo start 
```
# Technology Stack
* Expo CLI 3.18.2
* React Native 0.62
