# COVID-19-TRACKER
# INTRODUCTION
The aim of this project is to build a cross platform Covid-19 app to get live state-wise updates on the current situation of the pandemic.

# Prerequisities
```
node.js
expo
```


## To run this project, use Expo.
```
cd Covid-19-master
expo start 
```
# Technology stack
* Expo CLI 3.18.2
* React Native 0.62
