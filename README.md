# General info
A React Native covid tracking project with live updates on the current spread of the pandemic.

# Prerequisities
```
node.js
expo
```


## To run this project, use Expo.
```
$ cd Covid-19
$ expo start 
```
# Technologies
* Expo CLI 3.18.2
* React Native 0.62
